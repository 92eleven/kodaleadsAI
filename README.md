KodaLeads AI — README
Overview
A fully autonomous, self-improving AI agent that qualifies inbound leads and books appointments 24/7. Cloned and rebranded from a proven lead-qualification architecture.

Live demo
kodaleadsai.com — landing page showing the full customer-facing experience.

What it does
Capability	Detail
Lead qualification	GPT-4o, JSON structured output, configurable criteria
Appointment booking	Auto slot generation, instant confirmation
Self-improvement	Logs every interaction, weekly auto-tuning
Multi-channel	Gmail, SMS (Twilio), Google Calendar, webhook API
White-label	Rebrand in under 30 minutes
Deployment	Docker, one command
Quick start
git clone https://github.com/92eleven/KodaLeads.git
cd KodaLeads
cp .env.example .env
nano .env
nano knowledge_base.json
docker compose up -d
curl http://localhost:8080/health
